Segment Playbooks

每个产业节点对应的研究动作和模型入口。

EDA / IP / Design Enablement

Submarkets
EDA core tools; verification/emulation; analog/custom IC design; physical implementation; IP licensing; ASIC design services; silicon lifecycle analytics.
Drivers
Seat growth, token/cloud usage, maintenance renewal, IP royalties, hardware emulation backlog, design services NRE, China/export-control exposure.
Indicators
Advanced-node tape-outs, hyperscaler ASIC programs, foundry PDK releases, RPO/remaining performance obligations, emulation lead times, hiring in chip design teams.
Red flags
Large customer insourcing, regulatory delay around major M&A, China restrictions, EDA seat saturation, weaker design starts in handset/consumer.
First task
Build peer table separating software-like EDA from IP royalty and design-service models; map each Core company to node complexity and AI/custom-silicon exposure.

Compute / AI / Logic Chips

Submarkets
GPU/accelerators; server CPU; client CPU/GPU; FPGA/embedded; custom ASIC; DPU/NIC; edge AI SoC.
Drivers
Datacenter revenue, accelerator ASP, HBM/CoWoS allocation, software/networking attach, cloud capex, product cadence, gross margin by platform.
Indicators
Hyperscaler capex revisions, AI server lead times, HBM supply contracts, TSMC advanced-node mix, ODM backlog, accelerator benchmark adoption.
Red flags
Customer ROI fatigue, over-ordering, custom ASIC substitution, supply bottleneck moving outside chips, export restrictions, margin dilution from systems.
First task
Create AI compute scorecard: accelerator revenue, supply chain bottleneck, customer concentration, inference exposure, ecosystem lock-in and valuation sensitivity.

Mobile / RF / Connectivity

Submarkets
Mobile AP/modem; RF front-end; Wi-Fi/Bluetooth; display driver; image sensor; IoT connectivity; touch/fingerprint/audio chips.
Drivers
Handset units, premium mix, content per device, Android channel inventory, Apple content share, auto/IoT diversification, RF module integration.
Indicators
Smartphone sell-through, Taiwan fabless monthly revenue, China Android launch calendar, Apple build plans, distributor inventory, panel shipments.
Red flags
Apple insourcing, Android ASP pressure, inventory rebuild without sell-through, RF overcapacity, display driver commoditization.
First task
Split companies into handset-beta, connectivity-beta, display-beta and image-sensor-beta; do not compare RF, CIS and DDI on one multiple without cycle adjustment.

Networking Silicon / SerDes / Data Movement

Submarkets
Switch ASIC; routing silicon; optical DSP; retimer; SerDes IP; DPU/NIC; active electrical cable; CXL/PCIe fabric.
Drivers
Port-speed transition, switch radix, retimer attach rate, optical DSP ASP, custom silicon wins, cloud capex and AI cluster topology.
Indicators
800G/1.6T awards, Ethernet vs InfiniBand adoption, hyperscaler network architecture, Arista/Cisco order commentary, optical-module lead times.
Red flags
ASP erosion faster than port upgrade, cloud customer concentration, internal silicon displacement, CPO timing uncertainty, one-time pull-ins.
First task
Map every company to scale-out, scale-up, electrical or optical layer; build a port-speed migration table from 400G to 800G to 1.6T.

Analog / Power / Mixed Signal

Submarkets
Signal chain; power management; power discrete/modules; SiC/GaN; MCU; sensors; isolation; automotive radar/driver ICs.
Drivers
Industrial/auto inventory, channel sell-through, book-to-bill, utilization, pricing, SiC capacity, AI server power content.
Indicators
Distributor inventory days, auto production schedules, industrial PMI, EV inverter demand, data-center power density, lead times.
Red flags
Inventory correction masked by backlog, SiC price collapse, auto platform delays, China analog price competition, underloaded internal fabs.
First task
Separate long-cycle auto/industrial quality names from consumer analog and China localization names; normalize margins for utilization.

Memory / Storage Semiconductors

Submarkets
Commodity DRAM; HBM; NAND; enterprise SSD; HDD/nearline storage; NOR/specialty memory; controllers/modules.
Drivers
DRAM/NAND ASP, bit shipments, capex, inventory days, HBM qualification, enterprise SSD contract mix, storage nearline demand.
Indicators
Spot/contract memory pricing, memory capex plans, HBM supply agreements, hyperscaler storage orders, China smartphone/PC demand, module channel inventory.
Red flags
Annualizing peak ASP, HBM qualification loss, NAND overbuild, customer prebuy, capex discipline breakdown, hidden inventory at module makers.
First task
Build memory cycle sheet: ASP, bit growth, utilization, capex and HBM mix; value companies on normalized earnings and scenario margins.

Foundry / IDM Manufacturing

Submarkets
Leading-edge logic; mature foundry; specialty RF/BCD/power; IDM logic; specialty MEMS/sensor; captive advanced packaging.
Drivers
Utilization, wafer ASP, node mix, depreciation, capex, customer concentration, geographic capacity, subsidies, advanced packaging attach.
Indicators
Foundry monthly revenue, smartphone/PC sell-through, cloud ASIC tape-outs, WFE orders, fab utilization, node migration schedules.
Red flags
Mature-node overcapacity, depreciation cliff, technology slip, customer concentration, geopolitical disruption, underpriced government-led expansion.
First task
Separate advanced-node scarcity from mature-node cyclical recovery; build utilization and depreciation sensitivity by node and geography.

Semicap Equipment

Submarkets
Lithography; deposition; etch; clean; CMP; implant; thermal; track; compound semi tools; back-end equipment overlap.
Drivers
WFE TAM, orders, backlog, China mix, service revenue, installed base, node transition intensity, memory vs logic capex.
Indicators
TSMC/Samsung/Intel/Micron/SK capex, SEMI/WFE forecasts, equipment lead times, export-control updates, service growth.
Red flags
China pull-forward, order cancellations, memory capex pause, customer concentration, export restrictions, domestic substitution at lower margin.
First task
Build WFE wallet-share map by process step and customer; do not treat ASML/Lam/KLA/AMAT/TEL as interchangeable equipment beta.

Process Control / Test / Metrology

Submarkets
Metrology; inspection; ATE; probe cards; sockets/contactors; handlers; burn-in; system-level test; yield analytics.
Drivers
Test time, probe card ASP, advanced-node process-control intensity, HBM/AI complexity, new tester platforms, consumables refresh.
Indicators
AI/HBM product ramps, KLA/Advantest/Teradyne orders, OSAT capex, yield bottlenecks, new package qualification.
Red flags
One-platform demand spike, customer concentration, memory test pull-in, probe card pricing pressure, lower utilization at OSATs.
First task
Split process-control capital equipment from recurring probe/socket consumables; model AI/HBM test time as a separate driver.
